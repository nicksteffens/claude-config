# Example Skill

This is an example skill to demonstrate the structure and format.

## What This Skill Does

This skill demonstrates:
- Basic YAML frontmatter structure
- How to write skill instructions

## Usage

Claude will automatically invoke this skill when the user asks about examples or demonstrates skill capabilities.

## Instructions for Claude

When this skill is invoked:

1. **Explain the context** - Tell the user what triggered this skill
2. **Show capabilities** - Demonstrate what you can do
3. **Provide examples** - Give concrete examples
